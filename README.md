# Ecossistema End-to-End de Gestao e Analytics

Sistema de vendas: PostgreSQL + Flask + Bootstrap + Power BI.

## Como rodar
1. Criar base PostgreSQL e aplicar SQL em db/
2. Copiar .env.example para .env e preencher DATABASE_URL
3. pip install -r requirements.txt
4. cd app e python app.py
5. Abrir http://127.0.0.1:5000

Nunca publique o ficheiro .env.
